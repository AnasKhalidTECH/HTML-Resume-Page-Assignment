# HTML Resume Website

This project is a **single-page resume website built using only HTML**, created as part of a web development assignment.

The goal of this assignment was to practice:
- Proper HTML structure
- Semantic tags
- Lists, tables, and sections
- Clean and readable markup (without using CSS)
